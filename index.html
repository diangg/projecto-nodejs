Como criar um projecto node.js

Bom dia, Boa tarde ou Boa noite colegas. Hoje iremos criar o nosso primeiro projecto node.js de uma forma muito fácil e rápida. Bora mergulhar?!



Node.js é uma biblioteca da linguagem Javascript que permite criar aplicações web. Ele é caracterizado por ser uma linguagem muito fácil e rápida, feita do famosinho Javascript!

Todo projecto em node js é constituído por 5 pastas importantes são elas:

  • public 🌐
  • views 👀
  • routes 🛣
  • Controller 👨🏽‍💻
  • Models
